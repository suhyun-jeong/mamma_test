# Baby Food Porridge Ratio Calculator (이유식 배죽 계산기)

## Overview
A simple web-based calculator for determining the correct proportions of ingredients (rice, grains, meat, vegetables, and water) for different stages of baby food (porridge).

## Project Structure
- `index.html`: Main UI structure using semantic HTML.
- `style.css`: Modern styling with CSS variables and responsive design.
- `main.js`: Logic for calculating ingredient amounts based on the selected stage and desired total weight.

## Features
- **Stage Selection**: Choose between 10x, 7x, 5x, 3x, and 2x (thick rice) porridge stages.
- **Stage Information**: Displays the recommended age and a brief description for each stage.
- **Interactive Ratio Bar**: Visual representation of ingredient proportions for the selected stage.
- **Calculation**: Enter the desired total amount (50g to 1200g) to get exact ingredient weights and water volume.
- **Responsive Design**: Works on mobile and desktop browsers.

## Implementation Details
- **Web Standards**: Built with vanilla HTML, CSS, and JavaScript.
- **Modern CSS**: Uses CSS variables for consistent theming and flexbox for layout.
- **Dynamic Updates**: UI updates instantly when the stage is changed or a calculation is performed.
